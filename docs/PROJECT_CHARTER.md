## 2. Propósito y Justificación
*Centralizar la interacción académica y social de los estudiantes de la Universidad La Salle en una plataforma digital moderna y eficiente. Actualmente, la comunicación estudiantil suele estar fragmentada en múltiples grupos de mensajería instantánea. Este proyecto busca crear un espacio de discusión estructurado que permita el intercambio de conocimientos, anuncios y la creación de comunidades específicas, fomentando el sentido de pertenencia y la colaboración orgánica entre alumnos de diferentes ciclos y especialidades.*

## 3. Objetivos del Proyecto
1. Desarrollar una plataforma web que permita la gestión de perfiles de usuario, comunidades y publicaciones multimedia.
2. Implementar una arquitectura de base de datos híbrida, utilizando SQL para la integridad de los datos de usuario, y MongoDB para gestionar la jerarquía de los hilos de comentarios.
3. Integrar un Agente de IA que asista a los usuarios dentro de la plataforma.

## 4. Stakeholders
- Desarrolladores.
- Usuarios Finales. (Estudiantes, docentes y personal administrativo de la Universidad La Salle)

## 5. Alcance
- **Incluye**: Módulo de Usuarios, Módulo de Comunidades, Sistema de Publicaciones, Sistema de Interacción, identidad visual, notificaciones y Agente IA. 

- **No incluye**: Pasarelas de pago o sistemas de monetización, chat de mensajeria instantánea en tiempo real y aplicación móvil nativa.
